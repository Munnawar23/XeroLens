📸 XeroLens — Mobile App
Note: All code is located in the dev branch.

Built using Expo (React Native) + TypeScript

📱 CORE FEATURES
Vintage Camera Interface — Captures high-quality photos with an analog, retro feel.
Real-time Filters — Live preview of lens effects and vintage textures.
Photo Editing Suite — In-app tools to adjust and refine your captures post-shot.
Media Library — Organize and view your "Vintage Captures" directly in the app.
Favorite System — Mark your best shots for quick access 🧡.
High Performance — Smooth rendering using Skia for complex graphical overlays.
Sensory UI — Haptic feedback for shutter buttons and interactive elements for a premium feel.
Smart Permissions — Robust handling of Camera, Microphone, and Media Library access.

🏗️ TECH STACK & TOOLS
Category Library
Framework Expo (SDK 54)
Language TypeScript
Navigation Expo Router
State Management Zustand
Graphics/Shaders @shopify/react-native-skia
Animations React Native Reanimated + Lottie
Storage @react-native-async-storage/async-storage
Media Handling expo-camera + expo-media-library + expo-image-picker
Haptics expo-haptics
Notifications react-native-toast-message

🌈 THEMES & LOOK
Cinematic Aesthetics — Deep teal and dark themed UI (#3D6F72) for a studio-like atmosphere.
Immersive Display — Supports full edge-to-edge display for modern Android devices.
Optimized Performance — Uses Skia and Reanimated for 60fps animations and transitions.
